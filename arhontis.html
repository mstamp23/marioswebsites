Attention-Based RV Quantification

Deep learning pipeline for right ventricular volume estimation from 2D echocardiography using attention mechanisms. Associated with JASE 2024 paper.

# Attention-based RV Quantifier (2DE)
import torch
import torch.nn as nn

class RVAttentionNet(nn.Module):
    def __init__(self, in_ch=1, base=32):
        super().__init__()
        self.encoder = UNetEncoder(in_ch, base)
        self.attn    = SpatialAttention(base*8)
        self.head    = nn.Linear(base*8, 2)
    
    def forward(self, x):
        feats = self.encoder(x)
        feats = self.attn(feats)
        return self.head(feats.mean(dim=[2,3]))

Aortic Distensibility — ConvLSTM U-Net

Bi-directional ConvLSTM U-Net with dense connections for end-to-end spatiotemporal aortic lumen segmentation. Scientific Reports 2023.

# Bi-directional ConvLSTM U-Net
class BidirConvLSTMUNet(nn.Module):
    def __init__(self, T=30, ch=1):
        super().__init__()
        self.lstm_fwd = ConvLSTM(ch, 64, direction='fwd')
        self.lstm_bwd = ConvLSTM(ch, 64, direction='bwd')
        self.decoder  = DenseDecoder(128)
    
    def forward(self, video):
        # video: (B, T, C, H, W)
        h_f = self.lstm_fwd(video)
        h_b = self.lstm_bwd(video.flip(1))
        h   = torch.cat([h_f, h_b], dim=2)
        return self.decoder(h)

ACL Hybrid Novelty Detection

Reformulates ACL tear classification as novelty detection using aleatoric uncertainty to tackle class imbalance in MRI datasets. CMIG 2024.

# Aleatoric Uncertainty Novelty Score
def compute_novelty_score(logits, T=50):
    """MC Dropout uncertainty estimate."""
    preds = torch.stack([
        dropout_forward(logits) 
        for _ in range(T)
    ])  # (T, B, C)
    mean   = preds.mean(0)
    aleat  = -(mean * mean.log()).sum(-1)
    return aleat  # higher = more novel

Fast-Track ResNet for Arrhythmia

Leverages dynamical systems theory to accelerate deep residual network training for ECG arrhythmia classification. SITIS 2023.

# Dynamical Systems-Informed LR Schedule
class LyapunovScheduler:
    def __init__(self, opt, lambda_max=1.5):
        self.opt = opt
        self.lam = lambda_max
    
    def step(self, loss_grad_norm):
        # Adapt lr to Lyapunov exponent estimate
        lr = 1.0 / (self.lam * loss_grad_norm + 1e-8)
        for g in self.opt.param_groups:
            g['lr'] = lr.clamp(1e-5, 0.01)
        self.opt.step()
